Scala Modules Experiment

### Usage

Add this to your `build.sbt`:

```scala
libraryDependencies += "com.avast" %% "scala-modules-experiment" % "<version>"
```

Cross-builds are available for Scala 2.11.12, 2.12.8 and 2.13.0.
